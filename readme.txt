# Instructions to Start the Project

To start the project, follow these steps:

1. **Install Dependencies**  
   Run the following command to install all the necessary dependencies for the project:
   `npm i`

2. **Build the Project**  
   Once the dependencies are installed, build the project to prepare it for running:
   `npm run build`

3. **Start the Development Server**  
   Finally, start the development server to see the application in action:
   `npm run dev`

4. **Access the Application**  
   Open your browser and visit the following URL to view the application:
   `http://localhost:3000/`
